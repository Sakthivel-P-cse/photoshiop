# Photography Shop Website

A modern, responsive website for a photography shop, designed to be easily imported and customized in Webflow.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Interactive photo gallery with lightbox
- Services section with pricing
- Contact form
- Modern, clean design optimized for photography businesses

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with responsive design
- `script.js` - JavaScript for interactivity

## Importing to Webflow

1. Open your Webflow project
2. Go to the page where you want to add this content
3. Copy the HTML from `index.html` and paste it into a custom code embed
4. Copy the CSS from `styles.css` and add it to your project's custom code (Site Settings > Custom Code > Head Code)
5. Copy the JavaScript from `script.js` and add it to your project's custom code (Site Settings > Custom Code > Footer Code)
6. Replace placeholder images with your actual photos
7. Customize colors, fonts, and content to match your brand

## Customization

- Change the logo in the header
- Update the hero background image
- Replace placeholder images in the gallery with your portfolio photos
- Modify services and pricing
- Update contact information
- Adjust colors in `styles.css` to match your brand

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Troubleshooting

- If images don't load, check the image URLs
- For mobile issues, ensure viewport meta tag is present
- If JavaScript doesn't work, check console for errors
- For Webflow import issues, ensure custom code is added in the correct sections
